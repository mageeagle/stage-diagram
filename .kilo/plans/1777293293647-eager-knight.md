# Implement Custom StagePlan Node with Rotation and Resize Handles

## Task Description
The current StagePlan nodes are just standard `CustomNode`s which do not reflect the custom shape, rotation, or dimensions managed by the `StagePlanPropertiesPanel`. This task involves creating a dedicated `StagePlanNode` component that implements these visual properties and provides interactive handles for rotation and resizing.

## Steps

### 1. Create StagePlanNode component
- Create `src/components/nodes/StagePlanNode.tsx`.
- Implement rendering logic for:
    - **Shapes**: Rectangle (default), Circle (`rounded-full`), Triangle (`clip-path`).
    - **Transformations**: `rotation` (via `rotate()`), `width`, and `height`.
- Implement interactive handles (only visible when `selected`):
    - **Rotation Handle**: Positioned at the top; updates `rotation` in `useStagePlanStore`.
    - **Resize Handles**: Four corner handles; update `width` and `height` in `useStagePlanStore` by transforming mouse delta into local node coordinates.
- Integrate `Handle` components for inputs/outputs.
- Use `useUpdateNodeInternals` to keep React Flow in sync.

### 2. Register StagePlanNode in StagePlanCanvas
- Update `src/components/diagram/StagePlanCanvas.tsx` to use `StagePlanNode` for the `custom` node type.

### 3. Verification
- Test rotation handle interaction.
- Test corner resize handle interaction (including rotated nodes).
- Verify `StagePlanPropertiesPanel` updates match visual changes.
