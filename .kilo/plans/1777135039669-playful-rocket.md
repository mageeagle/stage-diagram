# Plan: Add Help Modal to Toolbar

This plan outlines the steps to implement a new "Help" modal that provides instructions on how to use the application, keyboard shortcuts, and creator credits.

## 1. Update State Management

Modify `src/store/useStore.ts` to include state and actions for the new help modal.

- Add `isHelpModalOpen: boolean` to `DiagramState`.
- Add `setIsHelpModalOpen: (isOpen: boolean) => void` to `DiagramState`.
- Initialize `isHelpModalOpen: false` in the store implementation.

## 2. Create Help Modal Component

Create a new component `src/components/help/HelpModal.tsx` following the pattern of `SettingsModal.tsx`.

### Components of the Modal:
- **Header**: "Help"
- **Description**: "This application provides a simple, unified interface for creating both Rider Lists and Signal Flows, helping you visualize and organize technical requirements more effectively."
- **Keyboard Shortcuts Module**: A section displaying a dummy list of shortcuts (e.g., `Ctrl+Z` -> `Undo`, `Ctrl+Y` -> `Redo`).
- **Credits Section**:
    - Name: Eagle Wu
    - Icon/Link to: https://www.qqaqq.net
    - Icon/Link to: [GitHub Placeholder]

### Implementation Details:
- Use `createPortal` to render at the end of `document.body`.
- Use Tailwind CSS for styling, maintaining consistency with existing components.
- Use `lucide-react` for icons (e.g., `HelpCircle`, `Github`, `User`, `X`).
- Handle the `Escape` key to close the modal.

## 3. Update Application Entry Point

Modify `app/page.tsx` to integrate the new modal.

- Import `HelpModal` from `@/components/help/HelpModal`.
- Extract `isHelpModalOpen` and `setIsHelpModalOpen` from the store using `useShallow`.
- Render `<HelpModal isOpen={isHelpModalOpen} onClose={() => setIsHelpModalOpen(false)} />`.

## 4. Update Toolbar

Modify `src/components/toolbar/Toolbar.tsx` to add the trigger button.

- Import the `HelpCircle` icon from `lucide-react`.
- Extract `setIsHelpModalOpen` from the store.
- Add a new `<button>` to the toolbar with the same styling as existing buttons, which opens the help modal when clicked.
