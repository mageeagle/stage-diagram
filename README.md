# Stage Diagram Editor

**Stage Diagram Editor** is a simple utility designed to help you visualize and organize technical requirements. It provides a unified way to manage **Rider Lists**, **Signal Flows**, and **Stage Plans** in one place.

## 🛠 How it works

The editor is split into three main areas to help you organize your technical documentation:

- **Signal Flow**: Use this to map out your connections and signal paths.
- **Rider List**: A list of your technical requirements. To save time, the Rider List can be **automatically generated** based on the nodes you have placed in your Signal Flow.
- **Stage Plan**: A visual layout of your stage setup. 

### Unified Nodes
Nodes are shared across all three areas. This means a node you define in your Signal Flow is the same one that appears in your Rider List and your Stage Plan, keeping your information consistent.

### Stage Plan Customization
While the other areas focus on connections and lists, the **Stage Plan** area allows for extra customization to help you match your physical setup. In this mode, you can adjust node **shapes** (rectangle, circle, triangle), **rotation**, and **dimensions** (width and height).

## 💡 Tips & Shortcuts

### Exporting
You can export your work to PDF or images. 
> **Note:** If your exported PDF is missing connection lines, please check that **"Print Background"** is enabled in your browser's print settings.

### Keyboard Shortcuts
| Shortcut | Action |
| :--- | :--- |
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `C` | Duplicate Selected |
| `Backspace / Delete` | Delete Selected |
| `Spacebar / Enter` | Create New |
| `Ctrl + Click` | Multiselect |
| `Shift + Drag` | Multiselect Area |
| `Esc` | Close Dialog |
| `Hold Q + Move Node` | Auto Connect Closest Node in Proximity |
| `Hold Q and A + Move Node` | Auto Connect Multiple Nodes in Proximity |
