# Plan: Add "Type" and "Location" properties to nodes

This plan outlines the steps to add two new properties, "Type" and "Location", to nodes. These properties will be selectable from lists managed in the `SettingsModal` and will include a "New..." option to add items on the fly.

## Implementation Steps

### 1. Update Types
- Modify `src/types/diagram.ts` to add `type?: string` and `location?: string` to the `CustomNodeData` interface.

### 2. Update Store (`src/store/useStore.ts`)
- Update `DiagramState` interface:
    - Add `updateNodeType: (nodeId: string, type: string) => void`.
    - Add `updateNodeLocation: (nodeId: string, location: string) => void`.
    - Update `addNode` signature to include `typeProperty?: string` and `locationProperty?: string`.
- Implement `updateNodeType` and `updateNodeLocation` to update the corresponding property in a node's `data`.
- Update `addNode` implementation to include `typeProperty` and `locationProperty` in the new node's `data`.

### 3. Update Node Creation (`src/components/diagram/NodeCreationModal.tsx`)
- Destructure `types`, `locations`, `addType`, and `addLocation` from `useStore`.
- Add local state for `selectedType` and `selectedLocation`.
- Add dropdown menus for "Type" and "Location" in the modal form.
- Implement "New..." option for both dropdowns:
    - Use `window.prompt` to get a new value.
    - If a value is provided, call `addType`/`addLocation` to update the global list and set the local state.
- Update `handleCreate` to pass the `selectedType` and `selectedLocation` to the `onCreate` callback.

### 4. Update Diagram Canvas (`src/components/diagram/DiagramCanvas.tsx`)
- Update `handleCreateNode` to accept `typeProp` and `locationProp` as arguments.
- Pass these new arguments to the `addNode` call.

### 5. Update Property Inspector (`src/components/inspector/PropertyInspector.tsx`)
- Destructure `types`, `locations`, `updateNodeType`, `updateNodeLocation`, `addType`, and `addLocation` from `useStore`.
- Add dropdown menus for "Type" and "Location" in the inspector.
- Implement "New..." option for both dropdowns:
    - Use `window.prompt` to get a new value.
    - If a value is provided, call `addType`/`addLocation` to update the global list and then call `updateNodeType`/`updateNodeLocation` to update the selected node.
