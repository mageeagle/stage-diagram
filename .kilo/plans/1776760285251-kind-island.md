# Plan for Node Creation Popup

## Goal
When adding a node, display a popup that asks for the initial number of inputs and outputs. The node should then be created with these counts, and outputs should be named "Output [Index]" starting from 1. The "Add Node" button should also trigger this popup.

## Implementation Steps

### 1. Update `src/store/useStore.ts`
- Add `isModalOpen: boolean` to `DiagramState`.
- Add `pendingPosition: { x: number; y: number } | null` to `DiagramState`.
- Add `setIsModalOpen: (isOpen: boolean) => void` to `DiagramState`.
- Add `setPendingPosition: (position: { x: number; y: number } | null) => void` to `DiagramState`.
- Update the implementation of `addNode` to generate nodes with custom input/output arrays based on provided counts.

### 2. Create `src/components/diagram/NodeCreationModal.tsx`
- Create a new component for the popup/modal.
- Use Tailwind CSS for styling.
- Include:
    - A title (e.g., "Create New Node").
    - Two numeric input fields: "Number of Inputs" and "Number of Outputs".
    - "Cancel" button to close the modal without creating the node.
    - "Create" button to call `addNode` with the specified counts and close the modal.
- Ensure the "Number of Inputs" field is automatically focused when the modal opens.
- Implement keyboard support:
    - `Enter` key should trigger the "Create" action.
    - `Esc` key should trigger the "Cancel" action.

### 3. Update `app/page.tsx`
- Update the `handleAddNode` function to set a default `pendingPosition` and set `isModalOpen` to `true` using the store actions, instead of calling `addNode` directly.

### 4. Update `src/components/diagram/DiagramCanvas.tsx`
- Remove local `isModalOpen` and `pendingPosition` states.
- Use `isModalOpen`, `pendingPosition`, `setIsModalOpen`, and `setPendingPosition` from `useStore`.
- Update the `keydown` event handler to use the store actions.
- Update `handleCreateNode` to use the store actions to close the modal and clear the pending position.

## Verification
- Pressing 'Space' or 'Enter' on the canvas should open a modal.
- Clicking the "Add Node" button should open the modal.
- Entering numbers in the modal and clicking "Create" should result in a node being added with the correct number of inputs and outputs.
- Outputs should follow the "Output [Index]" naming convention.
- Clicking "Cancel" should not create a node.
