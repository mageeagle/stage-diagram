# Plan for adding delete node functionality

## Goal
Implement a way to delete a node from the diagram, both via a button in the property inspector and by pressing the `Delete` key.

## Tasks

### 1. Update Store (`src/store/useStore.ts`)
- Add `deleteNode: (nodeId: string) => void` to the `DiagramState` interface.
- Implement `deleteNode` in `useStore`:
    - Remove the node from `nodes` array.
    - Remove all edges where the node is either the source or the target.
    - Clear `selectedNodeId`.

### 2. Update Property Inspector (`src/components/inspector/PropertyInspector.tsx`)
- Add a "Delete Node" button at the bottom of the inspector panel.
- The button should be visually distinct (e.g., red color) and use the `Trash2` icon.
- Clicking the button should call `deleteNode(selectedNodeId)`.

### 3. Update Diagram Canvas (`src/components/diagram/DiagramCanvas.tsx`)
- Add a keyboard event listener for the `Delete` key.
- If `Delete` is pressed and a node is currently selected, call `deleteNode(selectedNodeId)`.
- Use `useEffect` to manage the event listener.
