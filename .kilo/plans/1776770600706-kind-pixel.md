# Plan: Add Export Functionality to Diagram Canvas

This plan describes the steps to add a button that allows users to export the diagram canvas into various formats: PNG, JPG, SVG, and a vector-based PDF.

## Goals
- Implement an `ExportButton` component.
- Support exporting to PNG, JPG, SVG, and a vector-based PDF format.
- Integrate the `ExportButton` into the `DiagramCanvas` component.

## Implementation Steps

### 1. Install Dependencies
- Install `html-to-image` for capturing the SVG/DOM content.
- Install `jspdf` for generating PDF documents.
- (If required) Install `svg2pdf.js` to enable vector PDF generation from SVG.

### 2. Create `ExportButton` Component
- Create `src/components/diagram/ExportButton.tsx`.
- The component will:
    - Display a download icon button.
    - Show a dropdown menu when clicked, offering: PNG, JPG, SVG, PDF.
    - Use `lucide-react` for icons.
    - Accept a `ref` to the element that needs to be exported (the ReactFlow viewport).
- Implementation of export functions:
    - **PNG**: Use `toPng` from `html-to-image`, then trigger a download via a temporary link.
    - **JPG**: Use `toJpeg` from `html-to-image`, then trigger a download via a temporary link.
    - **SVG**: Use `toSvg` from `html-to-image`, then trigger a download via a temporary link.
    - **PDF (Vector)**: 
        1. Use `toSvg` from `html-to-image` to get the SVG content.
        2. Use `jsPDF` and `svg2pdf.js` to render that SVG content into a PDF document, preserving its vector nature.
        3. Trigger the PDF download.

### 3. Update `DiagramCanvas` Component
- In `src/components/diagram/DiagramCanvas.tsx`:
    - Use `useRef` to create a reference to the container wrapping the `ReactFlow` component or specifically the viewport element.
    - Wrap `ReactFlow` in a `div` with the `ref`.
    - Import and add the `ExportButton` component to the UI, positioned at the top-left (`top-4 left-4`) to avoid overlapping with `ThemeSwitcher` (`top-4 right-4`).
    - Pass the `ref` to `ExportButton`.

### 4. Verification
- Verify that each export format works as expected.
- Ensure the exported SVG and PDF files contain vector paths (test by zooming in on the exported files).
- Check for any UI issues or overlapping elements.
