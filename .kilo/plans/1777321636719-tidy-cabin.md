# Plan: Add Title, Subtitle, and Prepared By to Canvas and StagePlanCanvas

This plan implements the addition of `Title`, `Subtitle`, and `Prepared By` properties to both `DiagramCanvas` and `StagePlanCanvas`, and provides a way to edit these properties via the `SettingsModal`.

## 1. Update `useStore.ts` (Refactoring existing properties)
- Rename `title`, `subtitle`, `preparedBy` to `riderListTitle`, `riderListSubtitle`, `riderListPreparedBy` to distinguish them as part of the Rider List.
- Update corresponding update actions: `updateTitle` $\to$ `updateRiderListTitle`, `updateSubtitle` $\to$ `updateRiderListSubtitle`, `updatePreparedBy` $\to$ `updateRiderListPreparedBy`.
- Add new properties for `DiagramCanvas`: `canvasTitle`, `canvasSubtitle`, `canvasPreparedBy`.
- Add new update actions: `updateCanvasTitle`, `updateCanvasSubtitle`, `updateCanvasPreparedBy`.
- Update `restoreProjectState` to include the new `canvas...` properties and the renamed `riderList...` properties.

## 2. Update `DiagramCanvas.tsx`
- Retrieve `canvasTitle`, `canvasSubtitle`, and `canvasPreparedBy` from `useStore`.
- Add an overlay `div` to display these properties on the canvas.
  - Position: `absolute top-6 left-1/2 -translate-x-1/2 z-10` (centered at the top).
  - Styling:
    - Title: `text-2xl font-bold text-zinc-900 dark:text-white`
    - Subtitle: `text-lg text-zinc-600 dark:text-zinc-400`
    - Prepared By: `text-sm text-zinc-500 dark:text-zinc-500`
  - The text will be wrapped in a `div` with `text-center`.

## 3. Update `StagePlanCanvas.tsx`
- Retrieve `title`, `subtitle`, and `preparedBy` from `useStagePlanStore`.
- Add an overlay `div` to display these properties on the canvas, following the same styling as `DiagramCanvas.tsx`.

## 4. Update `SettingsModal.tsx`
- Import `useStagePlanStore` to access Stage Plan properties.
- Add three new sections to the modal: "Rider List", "Diagram Canvas", and "Stage Plan Canvas".
- Each section will contain three text input fields:
  - **Title**
  - **Subtitle**
  - **Prepared By**
- Bind the input values and `onChange` handlers to the respective stores:
  - For Rider List: `useStore` (`riderListTitle`, `riderListSubtitle`, `riderListPreparedBy` and their `update...` methods).
  - For Diagram Canvas: `useStore` (`canvasTitle`, `canvasSubtitle`, `canvasPreparedBy` and their `update...` methods).
  - For Stage Plan Canvas: `useStagePlanStore` (`title`, `subtitle`, `preparedBy` and their `update...` methods).
- Use a clear layout (e.g., sections with headings) to organize the 9 textboxes.

## 5. Update dependent components
- Update `src/components/toolbar/Toolbar.tsx` to use `riderListTitle`, `riderListSubtitle`, `riderListPreparedBy`.
- Update `src/components/diagram/node-list-modal/NodeListHeader.tsx` to use `riderListTitle`, `riderListSubtitle`, `riderListPreparedBy`.
- Update `src/components/diagram/NodeListModal.tsx` to use `riderListTitle`, `riderListSubtitle`, `riderListPreparedBy`.
- Update `src/components/diagram/node-list-modal/pdf-export-utils.ts` to use the new names if applicable.
- Update `src/utils/projectIO.ts` to include the new/renamed properties in `ProjectState`.

## Verification Plan
- Open the application.
- Check if `DiagramCanvas` shows the default canvas title/subtitle.
- Open `SettingsModal` and verify all 9 textboxes (3 for Rider List, 3 for Diagram Canvas, 3 for Stage Plan Canvas) are present and correctly populated.
- Edit a property in the `SettingsModal` and verify the change is reflected on the corresponding canvas or component.
- Switch to `StagePlanCanvas` and verify its properties are editable and displayed.
