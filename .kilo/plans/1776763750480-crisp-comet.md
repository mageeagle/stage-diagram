# Plan: Add a "highlight" for nodes when it's selected

This plan outlines the steps to implement a visual highlight for nodes in the diagram when they are selected.

## Overview

The goal is to provide visual feedback to the user indicating which node is currently selected. We will leverage the `selected` property provided by `reactflow`'s `NodeProps` within the `CustomNode` component and use Tailwind CSS to apply a highlight effect.

## Implementation Steps

### 1. Modify `src/components/nodes/CustomNode.tsx`

- Destructure the `selected` property from `NodeProps` in the `CustomNode` component.
- Update the container `div`'s `className` to conditionally apply a highlight style when `selected` is true.
- The highlight will consist of:
    - Changing the border color from `border-stone-400` to a more prominent color (e.g., `border-blue-500`).
    - Adding a ring/glow effect using Tailwind's `ring` utilities (e.g., `ring-2 ring-blue-200`).

## Verification

- Run the application (if possible in this environment, otherwise assume implementation is correct based on code review).
- Verify that clicking a node applies the highlight.
- Verify that clicking on the pane (background) removes the highlight from the node.
