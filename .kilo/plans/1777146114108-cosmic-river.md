# Implementation Plan: Display info text in PropertiesPanel for different Types/Locations

## Analysis

EdgeProperties.tsx shows info text when:
- Different Edge Types are selected (lines 102-106): "Multiple types selected"
- Different Cable Types are selected (lines 138-142): "Multiple types selected"

PropertiesPanel needs similar behavior for:
- Different Types (node type)
- Different Locations (node location)

## Implementation Plan

### Step 1: Add helper functions to detect mixed types/locations
- Inside PropertiesPanel, create `hasMixedTypes` function:
  - When isMultiSelect, collect all unique types from selected nodes
  - Return true if more than one unique type exists (excluding "none")
- Create `hasMixedLocations` function:
  - When isMultiSelect, collect all unique locations from selected nodes
  - Return true if more than one unique location exists (excluding "none")

### Step 2: Add display conditions for info text
- Type info: When isMultiSelect AND hasMixedTypes → show "Multiple types selected"
- Location info: When isMultiSelect AND hasMixedLocations → show "Multiple locations selected"
- Each info text shows independently based on its own condition

### Step 3: Add UI elements after Type selector (around line 124)
- Conditional render: {isMultiSelect && hasMixedTypes && ...info text}
- Text: <p className="text-[10px] text-gray-500 italic">Multiple types selected</p>

### Step 4: Add UI elements after Location selector (around line 157)
- Conditional render: {isMultiSelect && hasMixedLocations && ...info text}
- Text: <p className="text-[10px] text-gray-500 italic">Multiple locations selected</p>
- Style matches EdgeProperties: text-[10px], gray-500, italic
