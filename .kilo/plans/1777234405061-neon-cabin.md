# Plan: Add new "hideFromList" property for nodes excluded from NodeListModal

## Requirement Summary

Create a **new boolean property** on nodes specifically for excluding them from the NodeListModal report. This is separate from the existing `hidden` property which handles canvas visibility.

- **Purpose**: Hide nodes from the node list report while keeping them on the canvas
- **Name**: `hideFromList?: boolean`
- **Default**: `false` (new nodes appear in list by default)
- **Multi-select**: Updates all selected nodes with same value
- **Edge handling**: Exclude edges connected to hidden nodes from report


## Requirement Clarification

Based on user feedback: The existing `hidden` property should NOT be reused. Instead, create a **new boolean property** specifically for hiding nodes from the NodeListModal, while keeping the existing `hidden` property for its current purpose (canvas hiding).

           return {
             ...node,

       }),
     });
   },
