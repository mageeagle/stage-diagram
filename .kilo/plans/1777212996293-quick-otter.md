## Amended Step 4: Filter Edges in ExportButton.tsx (Replaces original Step 5)

**File:** `src/components/diagram/ExportButton.tsx`
**Location:** Lines 50-52 (`getNodesToHideForExport` function and surrounding area)

**Current behavior:** Only nodes are hidden before export via `prepareNodeForExport` which sets `exportingHidden: true`. Edges connected to those nodes are NOT hidden. The `DiagramCanvas.tsx` already hides edges when `edge.data.hidden` is true, but `prepareNodeForExport` doesn't update edge states.

**Problem:** When `prepareNodesForExport()` is called, nodes get `exportingHidden: true`, but edges connected to those nodes are NOT hidden. This means edges will still render during export even though their connected nodes are hidden.

**Solution:** Before calling `prepareNodesForExport()`, also temporarily hide edges connected to hidden nodes. After export, restore them.

### Changes Required

**Import `EdgeData` type:**
```typescript
import { CustomNodeData, EdgeData } from '@/types/diagram';
```

**Add helper function near `getNodesToHideForExport` (after line 52):**
```typescript
const getEdgesToHideForExport = (edges: XYFlow.Edge[], nodeIdsToHide: string[]) => {
  return edges.filter((edge) => {
    const edgeHidden = (edge.data as EdgeData)?.hidden ?? false;
    if (edgeHidden) return true;
    // Check if the edge is connected to any node being hidden for export
    const sourceHidden = nodeIdsToHide.includes(edge.source);
    const targetHidden = nodeIdsToHide.includes(edge.target);
    return sourceHidden || targetHidden;
  });
};
```

**Update `runExport` function (lines 64-117):**
Add edge hiding before the export, and restore after. The edges already have `display: none` applied via `DiagramCanvas.tsx` when `edge.data.hidden` is true, so we need to temporarily set edges to hidden during export.

**Alternative approach (simpler):** Since edges with `hidden: true` already render with `display: none` in `DiagramCanvas.tsx`, the visual exports (PNG/JPG/SVG/PDF) will automatically exclude them. However, we should verify that the `exportingHidden` mechanism for nodes also accounts for edges.

**Action needed in `runExport` (lines 64-117):**
```typescript
// Before prepareNodesForExport, also prepare edges
const nodesToHide = getNodesToHideForExport(nodes);
const nodeIdsToHide = nodesToHide.map((n) => n.id);

// Get edges connected to nodes being hidden - these need to be hidden too
const edgesToHide = edges.filter((edge) => {
  const edgeHidden = (edge.data as EdgeData)?.hidden ?? false;
  if (edgeHidden) return true;
  // Also check if connected nodes are being hidden
  const sourceNode = nodes.find((n) => n.id === edge.source);
  const targetNode = nodes.find((n) => n.id === edge.target);
  const sourceHidden = sourceNode?.data?.hidden ?? false;
  const targetHidden = targetNode?.data?.hidden ?? false;
  return sourceHidden || targetHidden;
});
```

**Add store actions for edge export hiding:**
Need to add `prepareEdgeForExport` and `restoreEdgeFromExport` actions in `useStore.ts` similar to `prepareNodeForExport` and `restoreNodeFromExport`.

## Amended Step 5: Add Edge Export Actions to useStore.ts

**File:** `src/store/useStore.ts`

### 5a. Add to interface (after line 87):
```typescript
prepareEdgeForExport: (edgeId: string) => void;
restoreEdgeFromExport: (edgeId: string) => void;
```

### 5b. Implement actions (after line 216):
```typescript
prepareEdgeForExport: (edgeId: string) => {
  set({
    edges: get().edges.map((edge) => {
      if (edge.id === edgeId) {
        return {
          ...edge,
          data: { ...edge.data, exportingHidden: true },
        };
      }
      return edge;
    }),
  });
},

restoreEdgeFromExport: (edgeId: string) => {
  set({
    edges: get().edges.map((edge) => {
      if (edge.id === edgeId) {
        return {
          ...edge,
          data: { ...edge.data, exportingHidden: false },
        };
      }
      return edge;
    }),
  });
},
```

### 5c. Update EdgeData type to include `exportingHidden`:
```typescript
export interface EdgeData {
  cableType: string;
  hidden?: boolean;
  exportingHidden?: boolean; // Temporary flag for export-time hiding
}
```

### 5d. Update DiagramCanvas.tsx edge hiding check (line 310):
The current check only looks at `edge.data?.hidden`. We need to also check `exportingHidden`:
```typescript
// Change from:
.filter((e) => e.data?.hidden)
// To:
.filter((e) => e.data?.hidden || e.data?.exportingHidden)
```
