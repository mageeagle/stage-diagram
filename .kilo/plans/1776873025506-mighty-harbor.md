# Plan: Add Import and Export buttons to app/page.tsx

This plan implements two new buttons in the main page to allow users to import and export their project state, similar to the functionality in the `SettingsModal`.

## Steps

1. **Modify `app/page.tsx`**:
    - **Imports**:
        - Add `exportProject` and `importProject` from `@/utils/projectIO`.
        - Add `Download` and `Upload` icons from `lucide-react`.
    - **State/Hooks**:
        - Destructure `nodes`, `edges`, `templates`, `types`, `locations`, and `restoreProjectState` from `useStore()`.
    - **Handlers**:
        - Implement `handleExport` by copying the logic from `SettingsModal.tsx:31`: `exportProject({ templates, nodes, edges, types, locations })`.
        - Implement `handleImport` by copying the logic from `SettingsModal.tsx:35`: `importProject(loadedFile).then((state) => restoreProjectState(state))`.
    - **JSX**:
        - Add a hidden `<input type="file" accept=".json" onChange={handleImport} className="hidden" />` to the component.
        - Inside the `div` containing the top-right controls (Undo, Redo, ThemeSwitcher, Settings), add:
            - An "Export" button with the `Download` icon.
            - An "Import" button with the `Upload` icon that triggers the hidden input.
        - Both buttons will use the existing button style used by Undo/Redo/Settings.

## Verification
- Verify that clicking "Export" downloads a JSON file.
- Verify that clicking "Import" opens a file picker and correctly restores the state upon selecting a valid JSON file.
- Ensure the buttons match the existing UI style.
