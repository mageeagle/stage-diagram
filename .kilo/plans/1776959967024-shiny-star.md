# Plan: Export Node List to PDF

## Goal
Add a button to the `NodeListModal` that exports the current list of nodes (aggregated) and cables to an A4 sized PDF document, ensuring the PDF matches the UI perfectly.

## Implementation Steps

1.  **Install dependencies**:
    *   `jspdf`

2.  **Create Report Generation Utility**:
    *   Create `src/components/diagram/node-list-modal/node-list-report-generator.ts`.
    *   Implement `generateNodeListReport(nodes, groupBy, edges)` which returns a structured array of "report rows" (Headers, Nodes, and Summary rows like "Power Sockets").
    *   This centralizes the logic used for both rendering the UI and generating the PDF.

3.  **Refactor `NodeListContent.tsx`**:
    *   Update `NodeListContent.tsx` to use `generateNodeListReport` to render the list. This ensures the UI and the PDF always stay in sync.

4.  **Create PDF Export Utility**:
    *   Create `src/components/diagram/node-list-modal/pdf-export-utils.ts`.
    *   Implement `exportToPdf(report: ReportStructure, cables: CableData)`.
    *   The utility will use `jspdf` to draw the report rows and the "Cables" section onto an A4 document.

5.  **Update UI**:
    *   **`src/components/diagram/node-list-modal-types.ts`**: Add `onExport?: () => void` to `NodeListHeaderProps`.
    *   **`src/components/diagram/node-list-modal/NodeListHeader.tsx`**: Add an "Export PDF" button (using `FileDown` icon).
    *   **`src/components/diagram/NodeListModal.tsx`**:
        *   Import the report generator and export utility.
        *   Generate the report structure and implement `handleExport`.
        *   Pass `handleExport` to `NodeListHeader`.

6.  **Verification**:
    *   Manually test the export button and ensure the PDF layout is correct.
