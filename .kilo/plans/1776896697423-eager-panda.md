# Plan for adding cable type labels to edges

This plan implements a custom edge component that uses `EdgeLabelRenderer` to display the cable type as a label on the edge in the diagram.

## Steps

1.  **Create `src/components/diagram/LabeledEdge.tsx`**
    *   Implement a custom edge component that wraps `SmoothStepEdge` from `@xyflow/react`.
    *   Use `getSmoothStepEdgePath` to calculate the midpoint coordinates for the label.
    *   Use `EdgeLabelRenderer` to render the `data.cableType` text.
    *   The label will be styled to be centered on the edge and have `pointer-events: none`.
    *   The label will only be rendered if `data.cableType` is present and is not `'none'`.

2.  **Update `src/components/diagram/DiagramCanvas.tsx`**
    *   Import the newly created `LabeledEdge` component.
    *   Replace `SmoothStepEdge` with `LabeledEdge` in the `edgeTypes` prop of the `ReactFlow` component.

## Verification
*   Check that edges with `cableType: 'none'` (the default) do not show any label.
*   Check that edges with a specific `cableType` (e.g., 'CAT6') show the label in the middle of the edge, using the `EdgeLabelRenderer` layer.
*   Verify that the label is correctly positioned and does not interfere with edge selection or interactions.
